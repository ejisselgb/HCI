Introduccion

En esta página aprenderás algunos conceptos básicos de HTML para que puedas crear tu propia página web. Primero se mostrará la sintaxis de algunas etiquetas para poner títulos y para ponerlos de distintos tamaños. Por ultimo, estarán unos pasos a seguir para empezar con tu primera página web.

Sintaxis títulos

  • < h1 > para título en negrita tamaño grande medio.
  • < h3 > para título en negrita tamaño medio.
  • < h2 > para título en negrita tamaño grande medio.
  • De igual forma hasta < h6>

Pasos a seguir

  1. Lo primero que tienes que hacer es utilizar tu editor de texto de preferencia.
  2. Crear un archivo .html.
  3. Escribe lo siguiente en el documento anteriormente creado: < !DOCTYPE html > < html >
